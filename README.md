
## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/Adib23704/Rental-Service--Nyntax.git
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Start the server:

   ```bash
   npm start
   ```

5. Open your web browser and navigate to http://localhost:8080 to access the application.
